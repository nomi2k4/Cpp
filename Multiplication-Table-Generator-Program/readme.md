# Multiplication Table Generator Program

A simple C++ program that generates a multiplication table for a given number.

## Overview
The program prompts the user to input a number, uses a for loop to generate the multiplication table, and prints the table with numbers from 0 to 10.

## Features
- User input for number  
- Looping structure using for loop  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
