# Simple Calculator Program

A simple C++ program that implements a basic calculator to perform arithmetic operations.

## Overview
The program prompts the user to input two numbers and an operator, performs the specified arithmetic operation using a switch statement, handles division by zero and invalid operator errors, and outputs the result.

## Features
- User input for numbers and operator  
- Basic arithmetic operations (+, -, *, /, %)  
- Error handling for division by zero and invalid operators  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
