# ASCII Value Finder Program Using Function

A simple C++ program that finds the ASCII value of a given character using a function.

## Overview
This program prompts the user to enter a character and uses the `getASCII` function to retrieve and display its ASCII value.

## Features
- User input for a character
- Uses a function `getASCII` to find ASCII value
- Clear and informative output

## Requirements
- C++ compiler
