# Range Sum Calculator Program

A simple C++ program that calculates the sum of all numbers within a given range.

## Overview
The program prompts the user to input two numbers (start and end of range), uses a while loop to iterate through the range, calculates the sum of all numbers, and displays the result.

## Features
- User input for range  
- Looping structure using while loop  
- Calculation of sum using accumulator variable  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
