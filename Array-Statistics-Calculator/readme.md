# Array Statistics Calculator

A simple C++ program that calculates and displays sum, maximum, and minimum for a set of 10 numbers input by the user.

## Overview
This program prompts the user to enter 10 numbers, stores them in an array, and calculates the sum, maximum, and minimum.  
It then displays the input numbers along with the calculated statistics.

## Features
- User input for 10 numbers
- Uses an array to store numbers
- Looping structures with `for` loops
- Conditional statements to find maximum and minimum
- Displays clear and informative output

## Requirements
- C++ compiler
