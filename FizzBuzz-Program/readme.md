# FizzBuzz Program

A simple C++ program that implements the classic FizzBuzz game.

## Overview
This program prompts the user to input a number and then prints numbers from 1 to the input number. Multiples of 3 are replaced with "Fizz", multiples of 5 with "Buzz", and multiples of both 3 and 5 with "FizzBuzz".

## Features
- User input for the upper limit
- Looping structure using `while` loop
- Conditional statements for FizzBuzz logic
- Clear and informative output

## Requirements
- C++ compiler (e.g., GCC)
- Standard C++ libraries (e.g., iostream)
