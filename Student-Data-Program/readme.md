# Student Data Program

A C++ program that manages student data, allowing the user to input and display information for multiple students.

## Overview
The program prompts the user to input student details such as name, roll number, total marks, and percentage, stores them in student objects, and displays the information.

## Features
- Takes student name, roll number, total marks, and percentage as input  
- Stores and displays student details for multiple students  
- Utilizes a class to represent student objects  
- Demonstrates the use of getters and setters to access and modify student information  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream, string)
