# Laptop Information Program

A simple C++ program that manages laptop information, allowing the user to store and display details about different laptops.

## Overview
The program allows the user to store attributes such as brand, model, serial number, color, price, processor speed, RAM, and screen size. It uses a class to represent laptop objects and demonstrates the use of getters and setters.

## Features
- Stores laptop attributes: brand, model, serial number, color, price, processor speed, RAM, and screen size  
- Utilizes a class to represent laptop objects  
- Demonstrates the use of getters and setters to access and modify laptop information  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream, string)
