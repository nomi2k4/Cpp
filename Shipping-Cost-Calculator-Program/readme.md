# Shipping Cost Calculator Program

A simple C++ program that calculates the shipping cost of a package based on its weight and distance.

## Overview
The program prompts the user to input the package weight and shipping distance, validates the distance, calculates the shipping cost using predefined rates, and displays the result.

## Features
- User input for package weight and shipping distance  
- Conditional statements for distance and weight validation  
- Predefined shipping rates based on weight categories  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
