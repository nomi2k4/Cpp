# Loudness Level Checker Program

A simple C++ program that checks the loudness level in decibels (dB) and prints a message describing the level.

## Overview
The program prompts the user to input the loudness level in decibels and outputs a message describing whether the sound is soft, moderate, loud, or very loud based on the input.

## Features
- User input for loudness level  
- Conditional statements to check loudness levels  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
