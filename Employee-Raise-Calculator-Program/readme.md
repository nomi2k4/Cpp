# Employee Raise Calculator

A simple C++ program to calculate an employee's raise based on their years of service.

## Overview
This program prompts the user to input their salary and years of service, calculates the raise percentage based on the years of service, and outputs the raise amount along with the new total salary.

## Features
- User input for salary and years of service
- Conditional statements to determine the raise percentage
- Clear and informative output

## Requirements
- C++ compiler
- Standard C++ libraries (e.g., iostream)
