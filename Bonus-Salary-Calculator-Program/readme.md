# Bonus Salary Calculator Program

A simple C++ program that calculates the total salary including bonus based on the employee's grade.

## Overview
This program prompts the user to input their salary and grade (5-20), calculates the bonus based on the grade, and displays the total salary including bonus.  
- Grade 15-20: 50% of salary  
- Grade 5-15: 25% of salary

## Features
- User input for salary and grade
- Uses `if-else` statements for grade-based bonus calculation
- Clear and informative output

## Assumptions
- Bonus structure:
    - Grade 15-20: 50% of salary
    - Grade 5-15: 25% of salary

## Requirements
- C++ compiler
