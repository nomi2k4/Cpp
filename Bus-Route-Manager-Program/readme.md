# BusRouteManager

A simple C++ program to calculate the total fare of multiple bus routes.

## Overview
This program allows users to input data for up to 6 bus routes, including the route name, total buses, distance, and fare.  
It then calculates and displays the total fare for all routes.

## Features
- User input for route data
- Calculation of total fare for multiple routes
- Display of total fare

## Requirements
- C++ compiler
- Basic understanding of C++ programming concepts
