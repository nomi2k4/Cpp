# Car Rental System

A simple C++ program that simulates a car rental system.

## Overview
This program defines a struct `Car` to store car data, including car ID, model, rental price per day, and availability.  
It allows users to view available cars, select a car to rent, and calculate the total rental cost.  
A discount of 10% is applied for rentals over 7 days.

## Features
- Displays available cars
- Allows users to select a car to rent
- Calculates total rental cost based on rental period
- Applies discount for long-term rentals (10% off for rentals over 7 days)

## Requirements
- C++ compiler (e.g., GCC)
- Standard C++ libraries (e.g., iostream)
