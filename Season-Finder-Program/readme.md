# Season Finder Program

A simple C++ program that determines the season based on the month of the year.

## Overview
The program prompts the user to input the month of the year, uses a switch statement to determine the season, and displays the corresponding season (WINTER, SPRING, SUMMER, FALL).

## Features
- User input for month  
- Switch statement for season determination  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
