# Simple Addition Program

A simple C++ program that performs an addition operation and displays the result in a formatted manner.

## Overview
The program takes two predefined integers, displays them along with the addition operator, calculates the sum, and shows the result.

## Features
- Predefined integers as input  
- Displays numbers and addition operator in a formatted manner  
- Calculates and displays the sum  
- Waits for user input before exiting  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)  
- `conio.h` for `_getch()` function (optional for pause before exit)
