# Character Counter Program

A simple C++ program that counts the number of characters in a user-inputted sentence.

## Overview
The program prompts the user to input a sentence, displays the input, and counts the number of characters in the sentence using the `length()` function.

## Features
- User input for sentence  
- `getline()` function for reading the entire sentence  
- `length()` function for counting characters  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream, string)
