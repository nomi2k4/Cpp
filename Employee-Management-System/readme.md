# Employee Management System

A simple C++ program to manage employee, student, manager, and scientist data.  
It allows the user to input and display information for each type of person.

## Overview
This program allows the user to collect and display information for employees (ID and name), students (school and degree), managers (title and dues), and scientists (number of publications).

## Features
- Collects and displays basic employee information (ID and name)
- Collects and displays student education information (school and degree)
- Collects and displays manager information (title and dues)
- Collects and displays scientist information (number of publications)

## Requirements
- C++ compiler
- Standard C++ libraries (e.g., iostream, string)
