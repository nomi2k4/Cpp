# Dynamic Shopping Cart Program  

A simple C++ program that implements a dynamic shopping cart where items can be added, and the cart automatically resizes when it reaches capacity.

## Overview  
This program allows users to add items (represented by integer IDs) into a shopping cart. When the cart becomes full, its capacity doubles automatically. Users can insert items and display all the items currently in the cart.

## Features  
- Insert items into the cart  
- Automatically resize the cart when capacity is reached  
- Display all items in the cart with their item numbers  
- Track current size and total capacity of the cart  

## Requirements  
- C++ compiler (e.g., GCC, MinGW)  
- Standard C++ library (`iostream`)
