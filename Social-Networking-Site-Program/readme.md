# Social Networking Site Program

A simple C++ program that simulates a basic social networking site using a doubly linked list to manage user posts.

## Overview
This program allows users to create posts with a message, privacy type, and date. Users can view posts one by one, view all posts, filter posts by privacy, edit existing posts, and delete posts while navigating through them.

## Features
- Create new posts with message, privacy, and date  
- Store posts using a doubly linked list  
- View posts one by one with next and previous navigation  
- Edit and delete existing posts  
- View all posts or filter posts by privacy type  

## Requirements
- C++ compiler (e.g., GCC, MinGW)  
- Standard C++ libraries (`iostream`, `string`)
