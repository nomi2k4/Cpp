# Month Days Identifier Program

A simple C++ program that identifies the months corresponding to a given number of days.

## Overview
The program prompts the user to input the number of days in a month, uses a switch statement to match the input with the corresponding months, and displays the result.

## Features
- User input for number of days  
- Switch statement for matching days with months  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
