# Character Case Converter Program

A simple C++ program that converts a given character from uppercase to lowercase or vice versa.

## Overview
This program prompts the user to input a character, checks if it is an alphabet, and converts it:  
- If uppercase, converts to lowercase  
- If lowercase, converts to uppercase  
It then displays the converted character.

## Features
- User input for a character
- Uses `isalpha`, `isupper`, `tolower`, and `toupper` for checking and conversion
- Clear and informative output

## Requirements
- C++ compiler
