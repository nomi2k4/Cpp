# Vehicle Purchase Cost Calculator Program

A simple C++ program that calculates the total cost of purchasing a vehicle by adding different types of taxes to the base price.

## Overview
This program prompts the user to enter the price of a vehicle, calculates sales tax, delivery tax, and property tax based on fixed rates, and then displays the total cost including all taxes.

## Features
- User input for vehicle price  
- Calculation of sales, delivery, and property taxes  
- Uses basic arithmetic operations  
- Displays the total purchase cost clearly  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
