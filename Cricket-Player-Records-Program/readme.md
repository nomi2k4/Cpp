# Cricket Player Records Program  

A C++ program that implements a singly linked list to manage cricket player records. Users can add player information and display all records. Players can be inserted at the head or tail based on their batting position.

## Overview  
This program allows users to maintain a list of cricket players with detailed statistics. Players designated as "opening" batsmen are added to the head of the list, while others are added to the tail. The list can store and display multiple details for each player.

## Features  
- Add a new player to the list  
- Automatically insert opening batsmen at the head  
- Insert other players at the tail  
- Display all player records  
- Store and display detailed statistics:  
  - Name  
  - Matches played  
  - Batting position  
  - Total runs  
  - Batting average  
  - Strike rate  
  - Total fours and sixes  

## Requirements  
- C++ compiler (e.g., GCC, MinGW)  
- Standard C++ library (`iostream`, `string`)
