# Invoice Management Program

A simple C++ program that manages invoice data, allowing the user to input and display information for multiple parts of an invoice.

## Overview
The program prompts the user to input part number, part name/description, quantity, and price per item. It stores and displays invoice details for multiple parts using a class to represent an invoice.

## Features
- Input for part number, part name/description, quantity, and price per item  
- Stores and displays invoice details for multiple parts  
- Utilizes a class to represent an invoice  
- Demonstrates the use of getters and setters to access and modify invoice information  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream, string)
