# Bank Account Withdrawal Program

A simple C++ program that simulates a bank account withdrawal system.

## Overview
This program defines a struct `Bankaccount` to store account data, including account number, holder name, and balance.  
It allows users to withdraw a specified amount while enforcing a minimum balance requirement of $500.

## Features
- Simulates bank account withdrawal
- Enforces minimum balance requirement of $500
- Displays remaining balance after withdrawal

## Requirements
- C++ compiler (e.g., GCC)
- Standard C++ libraries (e.g., iostream)
