# Cookie Calorie Calculator Program

A simple C++ program that calculates the total calories consumed based on the number of cookies eaten.

## Overview
This program prompts the user to input the number of cookies eaten and calculates the total calories consumed based on the assumption that each cookie has 75 calories.  
It then displays the result.

## Features
- User input for number of cookies eaten
- Uses multiplication operator for calorie calculation
- Clear and informative output

## Assumptions
- 1 bag of cookies = 40 cookies
- 1 bag of cookies = 10 servings
- 1 serving = 4 cookies
- 1 serving = 300 calories
- 1 cookie = 75 calories

## Requirements
- C++ compiler
