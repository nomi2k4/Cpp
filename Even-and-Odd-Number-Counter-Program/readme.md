# Even and Odd Number Counter Program

A simple C++ program that counts the number of even and odd numbers in a set of 10 integers input by the user.

## Overview
This program prompts the user to input 10 integers, stores them in an array, iterates through the array to count the number of even and odd numbers, and displays the results.

## Features
- User input for integers
- Array data structure for storing integers
- Looping structures using for loops
- Conditional statements for counting even and odd numbers
- Clear and informative output

## Requirements
- C++ compiler (e.g., GCC)
- Standard C++ libraries (e.g., iostream)
