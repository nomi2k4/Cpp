# Simple Employee Management Program

A simple C++ program that manages employee data, including employee number, name, and salary, for a predefined number of employees.

## Overview
The program takes input for employee number, name, and salary for each employee, stores the information using a class, and displays the employee details in a structured format.

## Features
- User input for employee number, name, and salary  
- Stores and displays employee information  
- Uses a class to represent employee data  
- Allows setting and retrieving employee details  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream, string)
