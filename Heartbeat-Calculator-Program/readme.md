# Heartbeat Calculator Program

A simple C++ program that calculates the approximate number of heartbeats in 2 years based on the user's input.

## Overview
The program prompts the user to input their heart rate per second, calculates the approximate number of heartbeats in 2 years by multiplying the heart rate by the number of seconds in 2 years, and displays the result.

## Features
- User input for heart rate per second  
- Calculates approximate heartbeats in 2 years  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
