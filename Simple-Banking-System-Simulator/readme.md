# Simple Banking System Simulator

A simple C++ program that simulates a basic banking system with options to view account balance, deposit money, and withdraw money.

## Overview
The program presents a menu with banking options, uses a switch statement to handle user input, and performs simulated transactions based on the user's choice.

## Features
- User input for menu choice  
- Switch statement for handling menu options  
- Simulated banking transactions (view balance, deposit, withdraw)  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
