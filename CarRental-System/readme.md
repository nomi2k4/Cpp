# Car Rental System

A simple C++ program that manages a car rental system. It allows adding cars and customers, renting cars to customers, and returning cars.

## Overview
This program provides functionality to manage cars and customers in a rental system.  
It allows adding new cars and customers, displaying available cars, renting cars to customers, and returning rented cars.

## Features
- Adds new cars to the rental system
- Adds new customers to the rental system
- Displays available cars
- Allows customers to rent cars
- Allows customers to return rented cars

## Requirements
- C++ compiler
- Standard C++ libraries (e.g., iostream, string)
