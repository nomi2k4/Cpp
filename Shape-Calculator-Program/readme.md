# Shape Calculator Program

A simple C++ program that calculates the area and perimeter of squares, rectangles, and circles based on user input.

## Overview
The program presents a menu to select a shape, prompts the user to enter the necessary dimensions, and calculates the area and perimeter of the selected shape.

## Features
- User input for shape selection and dimensions  
- Switch statement for handling multiple shape options  
- Calculates area and perimeter of squares, rectangles, and circles  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream, cmath)
