# Dynamic Grocery List Program  

A simple C++ program that implements a dynamic grocery list where items can be added, and the list automatically resizes when it reaches capacity.

## Overview  
This program allows users to add grocery items (represented as strings) into a grocery list. When the list reaches its capacity, it automatically increases by 5 slots. Users can insert items and display all the items currently in the grocery list.

## Features  
- Insert grocery items into the list  
- Automatically resize the list when capacity is reached  
- Display all grocery items  
- Track current size and total capacity of the list  

## Requirements  
- C++ compiler (e.g., GCC, MinGW)  
- Standard C++ library (`iostream`, `string`)
