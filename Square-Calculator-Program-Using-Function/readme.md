# Square Calculator Program Using Function

A simple C++ program that calculates the square of a given integer using a function.

## Overview
The program prompts the user to input an integer, calls a function `makedouble` to calculate the square of the input, and displays the result.

## Features
- User input for an integer value  
- Function `makedouble` to calculate the square of the input  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
