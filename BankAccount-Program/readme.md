# BankAccount Program

A simple C++ program that simulates a bank account, allowing users to check balance, deposit, and withdraw money.

## Overview
This program provides basic banking operations such as viewing balance, depositing money, and withdrawing money.  
It ensures simple input validation for deposits and withdrawals.

## Features
- Check account balance
- Deposit money
- Withdraw money
- Basic input validation for deposits and withdrawals

## Requirements
- C++ compiler
