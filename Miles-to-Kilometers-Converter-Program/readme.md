# Miles to Kilometers Converter Program

A simple C++ program that converts a distance in miles to kilometers.

## Overview
The program prompts the user to input a distance in miles, uses a conversion factor (1 mile = 0.6214 kilometers) to calculate the equivalent distance in kilometers, and displays the result.

## Features
- User input for distance in miles  
- Use of conversion factor for miles to kilometers conversion  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
