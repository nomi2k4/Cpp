# Pattern Printer Program

A simple C++ program that prints a numerical pattern.

## Overview
The program uses nested for loops to generate a pattern with 5 rows. Each row prints the current row number multiple times, increasing by 2 columns per row.

## Features
- Nested looping structure using for loops  
- Pattern generation based on loop counters  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
