# Dynamic Shopping Cart Program  

A C++ program that implements a dynamic shopping cart. Users can add items, remove items, search for items, and display the cart. The cart automatically resizes when the capacity is reached.

## Overview  
This program allows users to manage a shopping cart using dynamic memory allocation. The cart resizes automatically when full. Users can insert items, remove specific items, search for items, display all items, and clear the cart entirely.

## Features  
- Insert items into the cart  
- Automatically resize the cart when capacity is reached  
- Remove specific items from the cart  
- Search for an item by name  
- Display all items in the cart  
- Clear the entire cart  

## Requirements  
- C++ compiler (e.g., GCC, MinGW)  
- Standard C++ library (`iostream`, `string`)
