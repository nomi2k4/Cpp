# Shopping Cart Management Program

A simple C++ program that implements a shopping cart system using a doubly linked list to manage items.

## Overview
This program allows users to add items to a shopping cart, update item quantities, remove items, view cart contents, and generate a final bill. Each item contains details such as item ID, name, price, quantity, and total price.

## Features
- Add new items to the cart  
- Increase quantity of existing items  
- Remove items from the cart  
- Change quantity of items  
- View all items in the cart  
- Navigate items one by one  
- Calculate and print the total bill  

## Requirements
- C++ compiler (e.g., GCC, MinGW)  
- Standard C++ libraries (`iostream`, `string`)
