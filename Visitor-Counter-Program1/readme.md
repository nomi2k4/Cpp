# Visitor Counter Program

A simple C++ program that manages a list of website visitors by storing their IP addresses using a dynamic array.

## Overview
This program allows users to add visitor IP addresses, search for existing visitors, delete visitors, and display all stored IPs. The list dynamically resizes when capacity is exceeded.

## Features
- Add visitor IP addresses  
- Search for a visitor by IP  
- Delete a visitor IP  
- Display all visitor IPs  
- Dynamic resizing of visitor list  

## Requirements
- C++ compiler (e.g., GCC, MinGW)  
- Standard C++ libraries (`iostream`, `string`)
