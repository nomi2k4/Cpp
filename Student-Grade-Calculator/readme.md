# Student Grade Calculator

A simple C++ program that calculates a student's grade based on their marks in three subjects.

## Overview
This program defines a struct to store student data such as ID, name, and marks in three subjects. It calculates the average marks and assigns a grade based on the calculated average.

- Calculates average marks  
- Assigns grade based on average marks
