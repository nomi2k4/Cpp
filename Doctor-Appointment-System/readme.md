# Doctor Appointment System

## Description
This C++ program is designed to manage doctor appointments. It provides features for **admins**, **doctors**, and **patients**.

## Program Overview
- **Admins** can:
  - Add doctors  
  - View all appointments  
  - Show all doctors  

- **Doctors** can:
  - Edit their info  
  - Set their availability  
  - View their appointments  

- **Patients** can:
  - Show all doctors  
  - Book appointments  
  - Edit appointments  
  - Cancel appointments  

## Features
- User authentication for admins, doctors, and patients  
- Doctor management (add, edit, view)  
- Appointment management (book, edit, cancel, view)  
- Availability management for doctors  
- Clear and informative output  

## Functions Used
- `doctorLogin()` – Authenticates doctor login  
- `editDoctorInfo()` – Edits doctor information  
- `setAvailability()` – Sets doctor availability  
- `viewAppointments()` – Views doctor appointments  
- `BookAppointment()` – Books a new appointment  
- `editAppointment()` – Edits an existing appointment  
- `cancelAppointment()` – Cancels an existing appointment  

## Requirements
- C++ compiler
