# Marquee Booking and Billing Program

A simple C++ program that simulates a marquee booking system and calculates the total bill including GST.

## Overview
The program displays a menu of available marquees with their capacities and prices, prompts the user to select a marquee, time (day/night), and booking duration, and then calculates and displays the total bill including GST.

## Features
- User input for marquee selection, time, and booking duration  
- Conditional statements for calculating bill based on selected options  
- GST calculation and addition to total bill  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
