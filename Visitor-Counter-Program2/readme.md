# Visitor Counter Program  

A C++ program to manage a linked list of visitors based on their IP addresses. The program allows adding, searching, deleting, and displaying visitor records.

## Overview  
This program keeps track of visitors by storing their IP addresses in a linked list. Each visitor record contains an IP address, and the program supports operations such as adding new visitors, searching for existing visitors, deleting visitors, and displaying all visitor records.

## Features  
- Add a new visitor with an IP address  
- Search for a visitor by IP address  
- Delete a visitor by IP address  
- Display all visitor IP addresses  
- Keeps track of total visitor count  

## Requirements  
- C++ compiler (e.g., GCC, MinGW)  
- Standard C++ library (`iostream`, `string`)
