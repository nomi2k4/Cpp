# Character Incrementer Program

A simple C++ program that takes a character input from the user and displays the next character in the ASCII sequence.

## Overview
This program prompts the user to input a character, uses the prefix increment operator (`++a`) to increment the ASCII value, and displays the next character.

## Features
- User input for a character
- Uses prefix increment operator for character increment
- Clear and informative output

## Requirements
- C++ compiler
