# Temperature Control Program

A simple C++ program that simulates a temperature control system using a `heater` class.

## Overview
This program allows the user to control temperature by increasing or decreasing it through a menu-driven interface. The current temperature is displayed after each operation, and the program continues running until the user chooses to exit.

## Features
- Increase or decrease the temperature  
- Display the current temperature  
- Menu-driven user interaction  
- Continuous execution until exit option is selected  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream, string)
