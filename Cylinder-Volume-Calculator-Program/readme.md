# Cylinder Volume Calculator Program

A simple C++ program that calculates the volume of a cylinder based on user input.

## Overview
This program prompts the user to input the radius and height of the cylinder, calculates the volume using the formula V = πr²h, and displays the result.

## Features
- User input for radius and height
- Uses a `const` variable for π
- Uses multiplication operator for volume calculation
- Clear and informative output

## Requirements
- C++ compiler
