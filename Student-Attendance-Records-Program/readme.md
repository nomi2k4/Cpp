# Student Attendance Records Program  

A C++ program to manage a linked list of students, storing attendance details and allowing specific operations such as displaying the student with the highest attendance, searching by roll number, and removing a specific student.

## Overview  
This program allows you to manage a list of students with information including name, roll number, department, total classes held, classes attended, and attendance percentage. The program supports operations such as inserting students at the head or tail depending on attendance, displaying all records, finding the student with the highest attendance, searching by roll number, and deleting the third student in the list.

## Features  
- Add a new student to the list  
- Insert students at head if attendance > 80%, otherwise at tail  
- Display all student records  
- Display the student with the highest attendance  
- Search for a student by roll number  
- Delete the third student from the list  

## Requirements  
- C++ compiler (e.g., GCC, MinGW)  
- Standard C++ library (`iostream`, `string`)
