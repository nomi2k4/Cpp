# Bakery Billing System

A simple C++ program that calculates the total bill for a customer at a bakery, including GST.

## Overview
This program displays a bakery menu, takes the user's choice and quantity, calculates the subtotal, adds GST (16%), and displays the total bill.

## Features
- User input for item choice and quantity
- Uses `if-else` statements for item selection and calculation
- Calculates GST and total bill
- Clear and informative output

## Assumptions
- Item prices:
    - Coffee: Rs 200
    - Juice: Rs 100
    - Tea: Rs 50
- GST rate: 16%

## Requirements
- C++ compiler
