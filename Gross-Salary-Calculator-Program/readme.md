# Gross Salary Calculator Program

A simple C++ program that calculates the gross salary of an individual based on their working hours and hourly wage.

## Overview
The program prompts the user to input their working hours and hourly wage, calculates the gross salary by multiplying them, and displays the result in rupees.

## Features
- User input for working hours and hourly wage  
- Calculates gross salary  
- Displays the result clearly  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
