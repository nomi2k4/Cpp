# Age Comparison Program

A simple C++ program that compares the user's age with a fixed age value (20 years) and displays a message based on the comparison.

## Overview
This program takes the user's age as input and compares it with the programmer's age (20 years).  
Based on the comparison, it outputs whether the user is younger, the same age, or older.

## Features
- Takes age input from the user
- Compares user age with a fixed value (20)
- Uses conditional statements for comparison
- Displays clear output messages

## Requirements
- C++ compiler
