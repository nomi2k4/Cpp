# Ship Management Program

A simple C++ program that manages ship data, allowing the user to input and display information for multiple ships.

## Overview
The program allows users to enter ship names, number of passengers, and weight capacity, stores the data, and displays the ship details.

## Features
- User input for ship names, number of passengers, and weight capacity  
- Stores and displays ship details  
- Utilizes classes and objects to represent ships  
- Demonstrates the use of constructors, getters, and setters  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream, string)
