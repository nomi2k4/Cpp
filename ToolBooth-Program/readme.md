## ToolBooth Program

A simple C++ program that simulates a toll booth, calculating the total number of cars and the total money collected.

### Overview
This program simulates cars passing through a toll booth, distinguishes between paying and non-paying cars, calculates totals, and displays results.

### Features
- Simulates cars passing through a toll booth
- Distinguishes between paying and non-paying cars
- Calculates total number of cars and total money collected
- Displays totals and number of non-paying cars

### Requirements
- C++ compiler
- Standard C++ libraries (e.g., iostream, string)
