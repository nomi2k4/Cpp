# Odd and Even Number Printer Program

A simple C++ program that prints odd and even numbers within a given range, along with their squares.

## Overview
The program prompts the user to input a lower and upper limit, prints odd and even numbers within the range, and also prints their squares.

## Features
- User input for range limits  
- Looping structures using while loops  
- Conditional statements for checking odd/even numbers  
- Clear and informative output  

## Requirements
- C++ compiler (e.g., GCC)  
- Standard C++ libraries (e.g., iostream)
