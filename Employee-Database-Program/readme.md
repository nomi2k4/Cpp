# Employee Database Program

A simple C++ program that stores and retrieves employee data.

## Overview
This program defines a struct to represent employee data, including name, contact information, and salary.  
It prompts the user to input data for up to 10 employees and displays the names of employees who reside in Islamabad and have a salary between 50,000 and 80,000.

## Features
- Stores employee data in a struct
- Prompts user to input employee data
- Displays names of employees who meet specific criteria (Islamabad residence and salary range)

## Requirements
- C++ compiler
- Standard C++ libraries (e.g., iostream, string)
